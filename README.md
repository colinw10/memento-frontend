# Memento Frontend

A React frontend for the Memento microblog application.

## Quick Start

```bash
# Install dependencies
npm install

# Copy environment file
cp .env.example .env

# Start development server
npm run dev
```

## Team

See [docs/TEAM-README.md](docs/TEAM-README.md) for team assignments and task details.

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
